操作步骤：

  1. 初始化仓库:创建一个文件夹，cd入该文件夹后，执行git init
  2. 为远程仓库增加主机名：git remote add origin http://git-ma.paic.com.cn/papuhui/pamit_h5.git
  3. 拉取远程库的内容：git pull origin yuzuhang
  4. 创建本地分支：git branch yuzuhang001
  5. 更改当前分支为自己的本地分支：git checkout yuzuhang001
  6. 添加文件到暂存区：git add 文件.xxx
  7. 删除文件到暂存区：git rm 文件.xxx
  8. 提交说明：git commit -m "xxxxx"
  9. 推送到远程主机上对应的分支：git push origin yuzuhang001
